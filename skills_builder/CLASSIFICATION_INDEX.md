## Local Rule

Use the broad Dewey-style number for the shelf, then local decimals for this repo's working distinctions. Keep it useful. Do not turn it into numerology with a clipboard.

## Description Tag Rule

When a new skill is created, choose the Dewey-style code from the skill's main purpose and add it quietly to the end of the `description` field as `Dewey: xxx.xxx.`. Keep it small so the description still works as the trigger text.
